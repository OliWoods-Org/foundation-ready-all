> **Power outages disable life-sustaining medical equipment.** Shelters are often physically inaccessible. Deaf residents can't hear sirens. Blind residents can't read visual warnings. The intersection of disability + disaster is literally life-or-death, and zero tech solutions exist.

## The Solution

Personal emergency plans based on specific disabilities. Accessible alerts. Equipment backup coordination. Shelter accessibility database. Volunteer dispatching.

## Agents

**PrepPlanGenerator** — Personalized plan based on disabilities, equipment, mobility, communication needs
**AccessibleAlerts** — Multi-format alerts (text, vibration, visual, simplified language)
**ShelterFinder** — Real-time accessible shelter database with capacity
**EquipmentBackup** — Generator locations, battery exchange, oxygen supply tracking
**EvacRouteOptimizer** — Wheelchair-accessible routes, oxygen supply range, service animal needs
**VolunteerDispatcher** — Match helpers with specific assistance needs

## Safety

MANDATORY: Life-threatening emergencies → call 911. This tool supplements, never replaces, emergency services.

## Data Sources

FEMA disability data, ADA shelter requirements, NWS alerts, power grid APIs, Medicare DME records, transit accessibility data
